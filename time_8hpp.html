time an expression during evaluation.

This function is typically used for debugging purposes and it allows to check the numerical results of expressions and sub-expressions.

The following code times the intermediary results of the evaluation of \(\int_\Omega x y\)

auto e = Px()*Py();
std::cout << integrate( _range=elements(mesh), _expr=time(e,"e=") ).evaluate();
