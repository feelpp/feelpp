staticconstexpr

Returns a tuple of tuples containing the names and types of the basis fields supported by a mesh.

The function template takes a template parameter MeshType which is a type that represents a mesh. The function returns a tuple of tuples, where each inner tuple contains a string and a type. The string represents the name of a basis field type supported by the mesh, and the type represents the type of the basis field.

The function template is a static member function of the class that it is defined in. It is a constexpr function, which means that it can be evaluated at compile-time if the template parameter MeshType is a constant expression.

The function template uses a C++17 feature called structured bindings to create the tuple of tuples. It uses the hana library to create the tuples and the type_c function to create the type objects.

The function template is specialized for different types of meshes, depending on their dimension and the number of real dimensions. For example, for a 1D mesh with 1 real dimension, the function returns a tuple of tuples containing the names and types of the Lagrange basis fields with 1, 2, 0, and 1 DoFs. For a 2D mesh with 2 real dimensions, the function returns a tuple of tuples containing the names and types of the Lagrange basis fields (continuous order 1,2 and discontinuous 0,1 and vectorial continuous order 1,2 as well as Nedelec basis field 1st family of order 1)

Template Parameters

Returns
A tuple of tuples containing the names and types of the basis fields supported by the mesh.

Updates a field in a model mesh based on an expression and a range of elements.

This function template updates a field in a model mesh based on an expression and a range of elements. The function takes several parameters:

Parameters
nameA string representing the name of the field to be updated.
exprAn expression that represents the value to be assigned to the field.
rangeA range of elements over which the field should be updated.
basisA string representing the basis function space to be used for the field. If no basis is provided, the function will default to a scalar or vectorial function space based on the expression type.
initZeroA boolean flag indicating whether the field should be initialized to zero before being updated.

The function template uses Hana library to iterate over a tuple of supported basis function spaces for the given mesh type. For each basis function space, it checks if it matches the provided basis parameter.

  • If a match is found, it creates a function space of the appropriate type using the provided basis and checks if the field already exists in the mesh.
  • If the field does not exist, it is created using the function space.
  • If the field already exists but has a different basis, the field is rebuilt.
  • Finally, the function applies the expression to the field over the specified range of elements.
