print an expression during evaluation.

This function is typically used for debugging purposes and it allows to check the numerical results of expressions and sub-expressions.

The following code prints the intermediary results of the evaluation of \(\int_\Omega x y\)

auto e = Px()*Py();
std::cout << integrate( _range=elements(mesh), _expr=print(e,"e=") ).evaluate();

projection/interpolation of an expresion onto a noal functionspace

  • _space the function space to project onto
  • _range the range of mesh elements to apply the projection (the remaining parts are set to 0)
  • _expr the expression to project
  • _geomap the type of geomap to use (make sense only using high order meshes)
  • _sum sum the multiple nodal contributions if applicable (false by default)
