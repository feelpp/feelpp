Detailed Description

template<int M, int N = 1, int Order = 2>
struct Feel::vf::map_vector_field< M, N, Order >

defines a dictionary of vector fields

this data structure creates a dictionary of fields, it associates a string to a Ginac Expr of rank 1 and a set of vector. In the case of vector fields the size of the vector field must be given

auto e = expr("{x,y}:x:y");
auto z = expr("{0,0}:x:y");
map_vector_field<2> m { { "inlet", e }, { "wall", z } };
