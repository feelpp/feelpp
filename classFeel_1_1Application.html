void Feel::Application::run ( const double *  X,
unsigned long  P,
double *  Y,
unsigned long  N 
)
virtual

execute the set of Simget stored in the Application following the input/output model \( Y=F(X) \).

\( P\) is the number of inputs and \( N\) the number of outputs. Denote \( S \) (nSimgets()) the number of simgets stored in the Application. \( X \) and \( Y\) must be of size \( S P\) and \( S N \) respectively.
